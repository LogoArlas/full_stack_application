# Note Application

Project idea: note taking application

Description: A note taking application designed for users to type and store information.

Main Purpose: The purpose of the application is to take notes.
    Intended user: Anybody
    Entities: notes and user

How to use the application:
    Make an account with a username and password.
    Login with username and password.
    Create a note. 
    Type words, numbers, and special characters on the note.
    Save the notes and changes made to them.
    Navigation bar at the top of the screen.
    Logout of application.

Key features and functions:
    Add notes.
